Frequently Asked Questions

This is FAQ page.

Q: What is SAML BOX?

SAML BOX is a custom developed application deployed to help you test your SAML integration.

Q: Why do you deploy this platform?

From time to time, I want some test IDP and SP so that I can debug/troubleshoot my SAML integration, but couldn't find any useful ones online. Hence, I developed this website which could be helpful for others too. Plus, could be a good way to market ourselves.

Q: How to use SAML BOX?

First, you will need to have some understanding on the SAML protocols and then, your own app (either IDP or SP) to do SSO with SAML BOX

Then, go to Getting Started page to get started.

Q: What is SP or IDP landing page?

When you have uploaded either an IDP or SP metadata, a corresponding landing page for that entity is created, where you can browse a list of actions available, which is more convenient to manage.

Q: Am I able to view logs if I encounter any issue during SAML SSO?

Yes, go to either IDP or SP landing page, and there is view server log option.
